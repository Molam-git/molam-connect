# Molam UI (Design System)

Usage:
1. `npm install`
2. `npm run dev` — local dev with Vite
3. `npm run storybook` — visual catalog
4. `npm run build` — build dist (ESM + CJS + types)

Import in app:
```ts
import "molam-ui/dist/tokens.css";
import { Button } from "molam-ui";
```
# Molam UI (Design System)

Usage:
1. `npm install`
2. `npm run dev` — local dev with Vite
3. `npm run storybook` — visual catalog
4. `npm run build` — build dist (ESM + CJS + types)

Import in app:
```ts
import "molam-ui/dist/tokens.css";
import { Button } from "molam-ui";
```

## Composants inclus
- Boutons, cartes, entrées, modales, tableaux
- Topbar pour les apps Ops/Merchant
- Design tokens Tailwind + CSS variables light/dark
