# Molam Pay - Brique 9 API Factures

## Description
API pour le paiement des factures (eau, électricité, abonnements en ligne) via Molam Pay.

## Fonctionnalités
- Paiement gratuit des factures de base
- Multi-pays / multi-devise
- Intégration USSD/App/Web
- Sécurité renforcée (JWT, mTLS, HMAC)
- Réconciliation financière industrielle

## Installation

```bash
npm install
npm run migrate:dev
npm run dev
