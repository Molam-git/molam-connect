# Brique 23 - API Float Management

Gestion et optimisation de la trésorerie (float) immobilisée chez les banques partenaires.

## Objectifs

- Disponibilité (zéro rupture lors de retraits/versements)
- Coût minimal (frais bancaires, intérêts, FX)
- Risque maîtrisé (exposition par banque/pays)
- Conformité (plafonds réglementaires, limites internes)
- Réactivité (rebalancement automatique piloté par SIRA)
